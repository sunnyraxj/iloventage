# **App Name**: NextBazaar

## Core Features:

- Product Catalog & Browsing: Display products across a modern, responsive interface including a hero section, category browsing, featured collections, product listings with filtering, and detailed product pages.
- Shopping Cart & Checkout: Enable users to add products to a shopping cart, view cart contents, update quantities, and proceed through a multi-step checkout process.
- User Authentication & Dashboard: Secure user signup, login, and role-based access (customer/admin). Customers can view order history and track orders through a personalized dashboard.
- Secure Admin Dashboard: An exclusive administrative interface for managing products (add, edit, delete), categories, viewing all orders, and updating order statuses (Confirmed, Shipped, Delivered).
- AI Product Description Assistant: An AI tool within the admin panel that suggests and generates compelling product descriptions based on product details (name, category, keywords), aiding content creation for new or existing products.
- Integrated Payment Gateway & Order Creation: Integration with Razorpay for payment processing. Upon successful payment, an order is automatically created in MongoDB with an initial 'confirmed' status and linked to the user.
- Cloudinary Image Management: Seamless integration with Cloudinary for secure and optimized storage, retrieval, and delivery of all product images.

## Style Guidelines:

- The primary color is a refined indigo, embodying trustworthiness and a modern aesthetic, in a dark shade appropriate for a light scheme. HEX: #333399.
- The background color is a very light, desaturated indigo, providing a clean and subtle backdrop that maintains cohesion with the primary color. HEX: #F6F7FB.
- An accent color of vibrant cyan-blue, chosen to provide dynamic contrast and draw attention to calls-to-action or key interactive elements. HEX: #66B2E0.
- The 'Inter' font (sans-serif) will be used for both headlines and body text, ensuring a clean, modern, and highly legible experience across all content, consistent with the app's professional feel.
- Use clean, minimalist SVG icons throughout the interface to maintain a modern aesthetic and ensure scalability and fast loading performance. Icons should be easily understandable and support clear user interaction.
- Implement a responsive, grid-based layout for product displays and category sections, ensuring an optimal viewing experience across various devices. Emphasize clear spacing and intuitive content organization for enhanced usability.
- Incorporate subtle and performant micro-animations, such as hover effects on product cards or smooth transitions for cart updates and order status changes, to provide visual feedback and enhance the user experience without delaying load times.
